# GoogleAds
GoogleAds Werbung Analyse
📊 Projekt: Analyse des „Digital Adds“-Datensatzes
Dieses Projekt beinhaltet eine vollständige Analyse des „Digital Adds“-Datensatzes gemäß den unten stehenden Aufgabenstellungen:

🧩 Aufgaben und durchgeführte Analysen
Analyse fehlender Werte:
Für alle Variablen wurde eine Analyse fehlender Werte durchgeführt. Es wurde überprüft, ob Daten lückenhaft sind, und die Anzahl der fehlenden Einträge wurde pro Spalte bestimmt.

Ausreißeranalyse (Outlier Detection):
Für jede Variable wurden statistische Methoden wie der Interquartilsabstand (IQR) verwendet, um Ausreißer zu identifizieren und grafisch darzustellen.

Regressionsanalyse mit Visualisierung:
Lineare Regressionsmodelle wurden grafisch erstellt für:

SALES ~ GOOGLE

SALES ~ TIKTOK

Erstellung der allgemeinen Regressionsgleichung:
Es wurde eine multiple lineare Regressionsgleichung aufgestellt, um den Zusammenhang zwischen den Verkaufszahlen (SALES) und den vier Werbemedien zu modellieren:

SALES = a + b1·GOOGLE + b2·META + b3·TIKTOK + b4·INFLUENCER

Investitionsempfehlung bei einem Budget von 100 $:
Basierend auf den Regressionskoeffizienten wurde eine optimale Budgetverteilung auf die vier Medienkanäle ermittelt. Die Entscheidung wurde durch den Einfluss der jeweiligen Variable auf den Verkauf begründet.

Investitionsentscheidung basierend auf Retention & Einnahmen:
Bei einer jährlichen Retention Rate von 42 % und einem monatlichen Umsatz von 15 $ wurde die Rentabilität der Applikation beurteilt. Die Entscheidung wurde unter Berücksichtigung langfristiger Kundenbindung und ROI getroffen.
